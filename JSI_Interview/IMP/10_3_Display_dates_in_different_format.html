Displaying Dates in Different Formats

Date Function Descrption Example
getTime(): Number Returns the number of milliseconds elasped since midnight January 1,1970, universal time.

var date1 = new Date();
document.write(date1.getTime());
o/p: 1642959147465

toString(): String Returns a string representation of the day, date, and timezone in local time

var date1 = new Date();
document.write(date1.toString());
o/p: Sun Jan 23 2022 23:03:11 GMT+0530 (India Standard Time)

toUTCString(): String Returns a string representation of the day, date, and time in universal time

var date1 = new Date();
document.write(date1.toUTCString());
o/p:- Sun, 23 Jan 2022 17:33:11 GMT

toDateString(): String Returns a string representation of the day and date only, and does not include time or timezone

var date1 = new Date();
document.write(date1.toDateString());
o/p:- Sun Jan 23 2022

toTimeString(): String Returns a string representation of the time and timezone only, and does not include day or date

var date1 = new Date();
document.write(date1.toTimeString());
o/p:- 23:03:11 GMT+0530 (India Standard Time)

toLocalString(): String Returns a string representation of the day, date, time, given in local time.

var date1 = new Date();
document.write(date1.toLocalString());
o/p: 1/23/2022, 11:01:55 PM

toLocalDateString(): String Returns a string representation of the day and date only, given in local time.

var date1 = new Date();
document.write(date1.toLocalDateString());
o/p: 1/23/2022

toLocalTimeString(): String Returns a string representation of the time and timezone only, given in local time.

var date1 = new Date();
document.write(date1.toLocalTimeString());
o/p: 11:01:55 PM
