For In Loop

To traverse(or loop through) attributes of an object, we use for in loop.

objects represents real world entities
Ex: Car, Student, College, Employee, World, Company, etc..

var car={ model: 'Maruthi', numer: '0007', color: 'red' }

***Note: Objects contain collection of keys(attributes) and value(attribute value) Pairs
syntax: 
        for(var attribute in object){
            //statements
        }
