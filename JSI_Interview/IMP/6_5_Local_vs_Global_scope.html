Local vs Global Scope

scope indicates the accessiblity and visiblity of variables.
Local Scope:

1) Any Variable declared inside a function is considered as in local scope.

2) Can be accessible only inside that function where it is declared

3) Local variables will be avialable only in the function execution

4) Local variables are created as soon as the control enters the function body and deleted as soon as the control exists the function body

Global Scope:

1) Any Variable not declared inside a function is considered as Global scope.

2) Can be accessible any where in the script
