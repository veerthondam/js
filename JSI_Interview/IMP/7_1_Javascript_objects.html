Javascript Objects

Jasvascript is an Object based or prototype based Language.
i.e, in Javascript objects are created with the help of prototypes.

What is prototype ?
prototype is an object, which contains common functionalities that javascript object inherits.
I.e, When Javascript object is created, it inherits the common functionalities present inside a prototype object from which it is built.

Any Javascript Object is a collection of Properties and has a prototype.
Property: It is a key value pair.
where
key is the name of the property,
value is the value of the property and it can be data, an object or a function.

Data Property: attribute
If a property's value is a data, or an object then it is called Data property
It represents attribute of an object

Function Property: Behavior
If a property's value is a function then it is called function property or method
It represents behavior of an object


Note:**
1)  Javascript object is a collection of objects.
2)  Every Javascript object is a associated with a prototype from which it is built.
3)  All Javascript objects inherit their common functionalities from their prototype.
4)  Javascript objects are dynamic, i,e..,
we can add Properties to an object or delete properties from an object at run time.
