Creating Date Objects - constructor Notation

We create date objects using Date constructor Function.

1) var date1 = new Date();
Creates a date object using the system current date and time.

Ex:
var date1 = new Date();
document.write(date1); //object
document.write(date1.toString()); //string
document.write(date1.toUTCString()); //string

2) var date2 = new Date(date:String);
Creates a date object using the given date String.

Ex:
var date2 = new Date(Sun Jan 23 2022 22:08:42);
document.write(date2);

3) var date3 = new Date(year,month,date[,hours[,minutes[,seconds,[ms]]]]);
Creates a date object using the given date String.

Ex:
var date3 = new Date(2022,0,23,22,18,37);
document.write(date3);
