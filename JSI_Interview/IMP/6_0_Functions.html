Functions

A function is a block of code, meant to perform a specific task.
Whenever we want to perform some specific task, we create a function.


There are many ways we can create a function
1) Using Function Declartion.
2) Using Function Expression
3) Using Function Constructor
4) Using IIFF(Immediately Invokable Function Expression)

Syntax: 
        function function_name(parm1,parm2,...){
            //statements
        }    //by default a function returns undefined value.
        
        //call the function 

        function_name();
***Note: After return statement,Immediately it will jump to the end of the loop
