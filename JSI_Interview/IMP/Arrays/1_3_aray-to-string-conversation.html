Array To String Methods



Method Description Example

toString()

The toString() method returns a string with array values separated by commas.
The toString() method does not change the original array.

Syntax:
array.toString()

var arr = ["veer","SW", 50000];
display_data = arr.toString();
document.write(display_data); //veer,SW,50000

Array - join()

The Join() method returns an array as a String.

The Join() method does not change the original array

Any separator can be specified. The default is comma (,).

Syntax:

array.join(separator)

var arr = ["veer","SW", 50000];
display_data = arr.join();
document.write(display_data); //veer,SW,50000

display_data = arr.join("#"); document.write(display_data); //veer#SW#50000

Array - Concat() - Returns Array

The concat() method concatenates (Joins) two or more arrays.

It returns new array with containing th joined arrays.

It will not change the exisiting arrays.

Syntax:
array1.concat(array2, array3, ..., arrayX)

Ex:

var arr5 = [10,20,30,40,50];
var arr6 = [60,70,80,90];
var result = arr5.concat(arr6);
document.write(result);// 10,20,30,40,50,60,70,80,90

Array - Every()

The every() method executes a function for each array element.

The every() method returns true if the function returns true for all elements.

The arr.every() method checks whether all the elements of the array satisfy the given condition or not that is provided by a method passed to it as the argument.

Syntax:

arr.every(callback(element[, index[, array]])[, thisArg])

Ex:

/* every Method */
sum = 0;
function isEven(element, index,array){
if(element%2==0){
return sum += element;
