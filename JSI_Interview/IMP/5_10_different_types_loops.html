There are 3 different types of Loops
1. Linear Loop
2. Quadratic loop
3. Cubic Loop

Linear Loop: 
            var n=5
            for(var i=0;i<=n;i++){
                //statements    // n times
            }

Quadratic Loop: 
            var n=5;
            for(var i=1;i<=n;i++){
                for(var j<=n;j++){
                    //statements      //n x n times
                }
            }

Cubic Loop: 
            var n=5;
            for(var i=1;i<=n;i++){
                for(var j=1;j<=n;j++){
                    for(var k=1;k<=n;k++){
                        //statements      // n x n x n times
                    }
                }
            }

Dependent Quadratic Loop:
Inner Loop iterations depend upon the outer loop counter variable. 
        var n=5;
        for(var i=1;i<=n;i++){
            for(var j=1;j<=i;j++){
                //statements   // n(n+1)/2 times
            }
        }
