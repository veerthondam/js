Constructor Function

If we want to create multiple object of same type, then we take help of constructor Function

Constructor Function help us to define a constructor for any Object type.


Object type[class] is a combination of its constructor and prototype


Constructor is a special function responsible for constructing new object and called as soon as new object is created


Prototype is a special object contains shared properties and called behaviors of instances




Example 

        function rectangle(width, height){
            this.width = width;
            this.height = height;
            this.getArea = function(){
                return this.width*this.height;
            }
        }

        var rect1 = new rectangle(5,5);
        var rect2 = new rectangle(6,6);

In above
rectangle -- object type
function rectangle(width,height){ //somecode } -- constructor for rectangle object
rect1, rect2 -- objects or instances rectangle object type.
In background javascript engine creates
rectangle.prototype -- prototype of rectangle object type.
