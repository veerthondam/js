InstanceOf

The JavaScript instanceof operator is used to check the type of an object at the run time. It returns a boolean value(true or false).

Syntax:
var myVar = objectName instanceof objectType

Ex: 

        var names = ["raj","ranga","suptnik"];
    
        console.log(names instanceOf String) //false
        console.log(names instanceof Array) //true
        console.log(names instanceof Number) //false
