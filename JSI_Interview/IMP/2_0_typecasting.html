For Explicit type casting, we use three type of builtin functions

1) ParseInt()

2) ParseFloat()

3) eval()

1. ParseInt():

Converts a given value to an integer, if not possible it returns "NaN"

If a given value is a float type, then it discards the float part and returns the integer part of the number

If the given value is a string type, then it tries to extract and return the begining integer part, if not it will return "NAN (NOT A NUMBER)"
