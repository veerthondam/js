Object Literal Notation

Creating Objects using Object literal and object constructor are quite similar.
But there are some differences:

1) syntatic Difference
var rect1 = new Object();
var rect1 = {};

1) It Creates a new empty object - creates one memory location, called instance.

2) Sets the value of implicit prototype "__proto__"(reference variable - holds the address of another location) of a new empty object to its constructor type.

4) Returns the reference of new object
Note:

It is recommanded to use {}(object literal)
1) Execution speed is faster compared to object constructor.
2) Vulnerability in overriding Object constructor.
