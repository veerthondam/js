Constructor Function

Why Constructor Function ?
Object constructor and Object literal notations are convenient for creating single object.
If we want to create multiple objects of same type, we take help of Constructor function.

Note:
Creating multiple objects of same type using constructor function:

- Reduces Code redundancy

- Less time consuming

- Easy to manage



It is recommended for single object, if we want multiple objects, then go for constructor function 
            var rect = {
                width = 5;
                height = 5;
                getArea = function(){
                    return width*height;
                }
            }

if we want multiple objects, then go for constructor function 
                function rectange(width,height){
                    this.width = width;
                    this.height = height;
                    this.getArea = function(){
                        return this.width * this.height; 
                    }
                }
