Expression

A valid combination of operators and operands is known as Expression.

Ex: 2 + 2
2 * 2 + 2

Some Invalid Expression are:
Ex: 1) +2 2
2) * +2-+2

Operators

It is going to perform an operation on the given operands

There are three types:

1. Unary Operator

2. Binary Operator

3. Ternary Operator - conditional operator(?:)

1. Unary Operator

A unary opeator is an operator with only one operand. This operand comes either before or after the operator.
The unary operators are:

1. Unary plus (+)

2. Unary minus (-)

3. prefix/post fix increments (++)

4. prefix/posfix decrements(--)

1. Unary plus (+)
Ex: 
             let a = 10;
             a = +a; 
             a = -10;

*** In case you apply the unary plus or minus on a non-numeric value, it performs the same conversion as the Number() function. 

    let s='10';
    document.write(typeof s); 
    s=+s;
    document.write(typeof s);

    document.write(s);
    s=-s;
    document.write(s);

*** In case you apply the unary plus or minus on a boolean value, it shows 0 or 1 

        let f = false,
        t = true;
        console.log(+f); // 0
        console.log(+t); // 1

**** Pre/post increment increment and decrement 

        let  x = 10;
        let j = 20;
        c = --x+j++;
        console.log(c)
        console.log(j)
