Logical Operators - Used to connect one or more conditions

&& - Logical And Operator: - If both LHS & RHS operands are true then the result will true otherwise false.
|| - OR operator - Any one of operands will be true, it is true.
! - Logical Not operator - If the given operand is false, then the result will be true.
