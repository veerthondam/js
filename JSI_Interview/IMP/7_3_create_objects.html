Creating user Defined Objects:

Objects that are defined/created by the programmers are called as User defined Objects.

There are 3 Ways we can create Objects:
1) built-in object Constructor
2) Using Object literal Notation [short hand Notation of object constructor]
3) Constructor Function.


Note ***

Object Constructor and Object literal are convenient to create a single object.
If we need to create multiple similar objects then we take help of constructor Function.
