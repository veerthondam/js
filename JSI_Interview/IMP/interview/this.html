This Keyword - Regular Vs Arrow Functions

1)

console.log(this) - It will pointed to Window Object.

Scenario 1: Create one function like below:

var age = 23;
function displayAge(){
console.log(this) // window Object
console.log(this.age) //23
}
displayAge();

1) When we're adding variables and functions are going to the added to Window Object. In above age, display function is available under window Object.
2) When we call displayAge(), internally it will call like window.displayAge();
