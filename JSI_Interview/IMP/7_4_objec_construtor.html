Object Constructor

Object constructor is a constructor function of built-in object called Object
Object constructor is convenient to create a single object.


Syntax:
var ObjectName = new Object();
var college = new Object();

1) It Creates a new empty object - creates one memory location, called instance.

2) Sets the value of implicit prototype "__proto__"(reference variable - holds the address of another location) of a new empty object to its constructor type.

3) Calls its constructor function with "newely created memory location address" with "list of arguments" if passed.

4) Returns the reference of new object, if the constructor function doesnot return any thing explicitly.
