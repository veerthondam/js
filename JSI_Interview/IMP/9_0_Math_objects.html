Contextual Classes

Contextual classes can be used to color table rows or table cells. The classes that can be used are: .active, .success, .info, .warning, and .danger.

Math Function Descrption Example
sign(value:Number): Number

Returns sign of the given value, If the given value is

+ve returns -> 1

-ve returns -> -1

else it returns -> 0

document.write(Math.sign(10)) //1

document.write(Math.sign(22)) //1

document.write(Math.sign(-34)) //-1

document.write(Math.sign("a")) //NaN

document.write(Math.sign(0.023)) //1

document.write(Math.sign(0)) //0

abs(value:Number): Number Returns the absolute value(+ve) of the given value.

document.write(Math.abs(5)) //5

document.write(Math.abs(-8)) //8

document.write(Math.abs(0.897)) //0.897

document.write(Math.abs(-0.897)) //0.897

sqrt(value:Number): Number Returns the square root of the given value.

document.write(Math.sqrt(4)) //2

document.write(Math.sqrt(16)) //4

document.write(Math.sqrt(64)) //8

pow(value1:Number,value2:Number): Number Returns the result of the value1(base) raise to the power of value2(exponent)

document.write(Math.pow(8,2)) //8*8 = 64

document.write(Math.pow(7,2)) //7*7 = 49

document.write(Math.pow(8,4)) //8*8*8*8= 4,096

max(value1:Number,value2:Number,....valueN: Number) Returns the maximum of zero or more values document.write(Math.max(8,4,10,40,60)) //60
min(value1:Number,value2:Number,....valueN: Number) Returns the maximum of zero or more values document.write(Math.min(8,4,10,40,60)) //4
round(value:Number):Number Rounds the given value to the nearest integer and return it
For Ex:2.2
[ in the decimal part 0-4] it return 2
whereas [decimal part 5-9] it returns 3

document.write(Math.round(8.4)) //8

document.write(Math.round(8.7)) //9

ceil(value:Number): Number Returns the upward number, irespective of decimal part.
Ex: Math.ceil(3.5)
No matter of decimal part, it will take to the top number like 4.

document.write(Math.ceil(4.1)) //5

document.write(Math.ceil(4.8)) //5

floor(value:Number): Number Returns the downward number, irespective of decimal part.
Ex: Math.floor(3.5)
No matter of decimal part, it will take to the nearest small number like 3.

document.write(Math.floor(7.3)) //7

document.write(Math.floor(7.9)) //7

trunc(value:Number):Number Excludes the decimal part and returns the integer part
For Ex:2.8
removes .8 decimal part and return 2 integer part

document.write(Math.trunc(9.4)) //9

document.write(Math.trunc(9.9)) //9
