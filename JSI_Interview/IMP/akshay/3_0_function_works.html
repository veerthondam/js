How Js Function will work ?

If two functions have same variable 'x', how it behaves

whenever a js program executed, it created a Global Execution context

It will have two components like:
1) Memory Component [phase -1 ] - Also known as Variable Environment
2) Code Component [phase - 2 ]

Before execution the single line of code, Memory will allocated in the first phase

Phase - 1 [ Memory Phase ]

Steps: [based on below Program]

1) It reserved Memory for
x - undefined
a() - {...} [function code]
b() - {...} [function code]
This will Memory pushed into call Stack, Now Global Execution Stack is inside the Call Stack

Phase - 2 [ code execution Phase ]

Steps: [based on below Program]

1) It runs
x = 1 ,undefined 1
a() - {...} [function code] - Here, again execution context will be created,limited to that functionwhenever execution context is created, it is pushed into callstack. After completed execution [execution context]Memomory&code is vanished and it is poped out from the call stack
b() - {...} [function code]
This will Memory pushed into call Stack, Now Global Execution Stack is inside the Call Stack
