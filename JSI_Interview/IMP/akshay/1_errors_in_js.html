Errors in Javascript

There are 3 types of Errors in Javascript

1) Syntax Error -> Uncaught SyntaxError: Missing initializer in const declaration
For Ex:
const b;
b=100;
2) TypeError -> Assignment to constant
const b = 100;
b=1000; //Error
3) Reference Error: cannot access "z" before initialization. 
            console.log(z); //ReferenceError: Cannot access 'z' before initialization
        let z=10;
