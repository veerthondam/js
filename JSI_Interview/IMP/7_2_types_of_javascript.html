Types of Javascript Objects

There are two types of Objects
1) Built in objects
2)  user Defined Objects

1) Built in Objects:
Objects that are predefined in the language and are responsible for handling specific task
Ex:
i)  Common reference objects:
Number, String, Math, Date, RegEx, Array, Function, Object

ii)   DOM(Document object Model) Reference objects:
Document, Elements, Attributes, Events, Style

iii)   BOM(Browser object Model) Reference objects:
Window, Navigator, Screen, History, Storage

iv) Other Reference Objects
Global, Error, Conversion, Storage, Arguments, Statements
