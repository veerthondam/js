Adding/Deleting Properties to an Object

Javascript Objects are dynamic in nature. I,e. we can add properties to an object at run time.

Syntax:

objectname.PropertyName = value //dot notation.(where value can be a single data, an object, or an function. )

objectname["PropertyName"] = value //array notation.

Ex:
rect1.width = 5; // data.property : attribute

rect1.height = 5; // data.property : attribute 

rect1.getArea = function(){ // function property : method or behavoir
 return this.width * this.height;
}


Delete an Obect

Syntax:
delete ObjectName.propertyName //dot notation
delete ObjectName["propertyName"] //array notation
