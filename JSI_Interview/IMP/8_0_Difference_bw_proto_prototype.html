Prototype Object

Prototype is a special object meant to hold shared attributes and behavoirs of instances

Syntax:
objectTypeName.prototype.sharedPropertyName=value; 

           function Rectangle(width,height){
               this.width = width;
               this.height = height;
               this.getArea = function(){
                   return this.width * this.height;
               }
               Rectangle.prototype.countV++;
           }

           Rectangle.prototype.countV = 0;
           var rect1 = new Rectangle(5,5);
          document.write(rect1.getArea());
We can access our countV variable
document.write(rect1.countV);
