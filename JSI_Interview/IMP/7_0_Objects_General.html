Objects

What is an Object in general

object represents any real world entity
Ex: car, company, student, college, Employee, etc..


object are going to have attributes and behaviors

Attributes:
   Attributes is a property of an Object.
Ex: color, model, name, address, rollnumber, etc..


Behavior:
   behavior is the functionality of an Object.
(What an Object can do, or what we can do to the Object)

Ex: Start, Stop, paint, getdistance, etc..
