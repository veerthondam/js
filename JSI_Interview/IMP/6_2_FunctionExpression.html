Function Expression

The difference between Function declartion and function expression is 1) we can call a function before the function definition, where as it is not possible in function expression. 
        1) Function Declartion

        sayHi() //there is no error
    
        function sayHi(){
            document.write("Hello")
        }

        2) Function Expression

        sayHi() //Error
    
        function sayHi(){
            document.write("Hello")
        }
